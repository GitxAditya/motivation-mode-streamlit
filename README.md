# ⚡ Motivation Mode

A simple motivational web application built with **Streamlit** that displays inspiring quotes and allows users to play their favorite background music while reading them.

This is one of the very first Streamlit projects I built during my learning journey.

Although it's a small and beginner-friendly project, I'm keeping it on my GitHub as a reminder of where I started. Every developer has a first project, and this is one of mine.

---

## Features

- Random motivational quotes
- Generate a new quote with one click
- Displays today's date
- Upload and play background music (MP3/WAV/OGG)
- Simple and clean Streamlit interface

---

## Tech Stack

- Python
- Streamlit

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/motivation-mode-streamlit.git
```

Move into the project

```bash
cd motivation-mode-streamlit
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
